🌐 Real-World Application

{% if algorithm == 'BFS' %} Emergency & Evacuation Routing BFS explores layers uniformly, guaranteeing the path with the fewest steps. In flood evacuations or building fires, BFS is used to map the fastest escape route to safety where every second counts and all passable corridors are equally safe. {% elif algorithm == 'DFS' %} Subterranean Cave & Pipe Exploration DFS explores deep along a single path before backtracking. This is used by autonomous mapping drones in unknown subterranean caves, collapsed mines, or complex sewer pipe networks where exploring corridors to the end is preferred. {% elif algorithm == 'ASTAR' %} GPS Vehicle Navigation (e.g. Google Maps) A* uses heuristics (estimated distance to destination) to dramatically speed up pathfinding. This is the foundation of modern GPS navigation systems, logistics routing, and video game pathfinding to find the shortest obstacle-free route. {% elif algorithm == 'UTILITY' %} Hazardous Goods Transport (Hazmat Shipping) Utility Search balances path length against risk penalty (Utility = Cost + W * Danger). This is used in Hazmat logistics and military convoy routing to decide if taking a longer, safe bypass is better than a direct but high-risk route. {% elif algorithm == 'CSP' %} Restricted Drone Flight Path Planning CSP treats safety as a hard constraint (e.g., drone must never enter zones with >50% wind turbulence and must remain under battery step budget). If no path satisfies all conditions, it halts flight instead of risking a crash. {% elif algorithm == 'MDP' %} Robotic Mars Rover / Icy Road Navigation MDP models non-deterministic action uncertainty (80% move success, 10% slide risk) and computes optimal policies. This is used by Mars Rovers and self-driving cars on icy roads where actions are slippery and safety must be maximized dynamically. {% endif %}
